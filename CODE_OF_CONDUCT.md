## Code of Conduct

This project is meant to be useful, practical, and welcoming.

If you open an issue, suggest an improvement, or submit a pull request, please assume good intent and help keep the conversation constructive.

We want this repository to be a place where people can:

- ask basic questions without feeling out of place
- share real implementation details, not just ideas
- disagree about tradeoffs without turning it personal
- improve the project together

Unacceptable behavior includes harassment, insults, personal attacks, deliberate intimidation, doxxing, or publishing private information.

If you notice a problem, open a private report through GitHub or contact the maintainer through the repository profile.

The maintainer may remove comments, issues, or contributions that make collaboration harder or less safe for others.
